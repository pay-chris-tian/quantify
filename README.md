# 🧠 퀀트 투자 자동화 프로젝트: 위험 회피형 가치 기반 전략

## 📌 프로젝트 개요

이 프로젝트는 **재무제표 기반의 저평가 종목을 선별하여, 위험을 최소화하면서 안정적인 수익을 추구하는 퀀트 전략 시스템**을 개발하는 것을 목표로 합니다.  
감정적 개입 없이 규칙 기반으로 의사결정을 내리고, 경험과 기술을 축적하며 점진적으로 실전 투자에 이르는 것을 지향합니다.

## 🎯 프로젝트 목표

### 단기 목표 (1단계: 3~6개월)
- 가치 기반 투자 전략 설계 및 POC 개발
- 월 단위 수익률 +1% ~ +3%, 손실 -3% 이내 유지
- 최소 3가지 전략 테스트 및 리포트화
- 백테스트 기반 실험 (실전 투자 전 단계)
- Streamlit 기반 대시보드 또는 자동화된 보고서 구축

### 중장기 목표 (2단계 이후)
- 월 수익률 **5~7%** 달성 가능한 전략 정립
- 실전 매매 자동화 시스템 구축 (API 연동)
- 리스크 관리 전략 포함 (손절, 익절, 자산 분산 등)
- 가족을 위한 실질적 자산 관리 수단 확보

## 🧬 투자 철학 및 전략 원칙

| 항목 | 내용 |
|------|------|
| 투자 성향 | 위험 회피형, 관망형 → 점진적 전환형 |
| 핵심 철학 | 감정 배제, 규칙 기반 매매, 학습 중심 |
| 선호 전략 | 가치 기반 (PER/PBR/ROE 등), 퀀트 필터링 |
| 리스크 관리 | 손절선/익절선 설정, 분산 투자 |
| 기대 방식 | 빠른 수익보다 **지속 가능성**과 **안정성** |

## 🛠 기술 스택 및 도구

| 용도 | 도구/기술 |
|------|------------|
| 데이터 수집 | `requests`, `pandas`, `BeautifulSoup`, `OpenDART`, `Naver Finance` |
| 전략 개발 | `pandas`, `numpy`, `TA-Lib`, `scikit-learn` (옵션) |

| 실거래 연결 | 키움 Open API+, NH 투자증권 API 등 (추후 적용) |
